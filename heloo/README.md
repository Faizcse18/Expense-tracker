# Expense Tracker Project

## Description
This is a Django-based Expense Tracker web application that helps users manage their personal finances. Users can sign up, log in, add expenses, set budgets, track savings goals, and view analytics for better financial management.

## Features
- User Authentication: Sign up, login, and logout
- Expense Management: Add, update, delete, and view expenses
- Budget Management: Set budgets, track spending, see budget status
- Savings Goals: Create and manage savings goals
- Analytics Dashboard: View total spending, category breakdown, daily and weekly trends
